# Espresso Hosting Website Redesign

This project is a complete redesign of the Espresso Hosting website, built with Next.js, TypeScript, and Tailwind CSS. The goal is to create a fresh, modern, and visually stunning website while maintaining all existing functionality and content.

## Project Overview

- **Next.js (latest version) with TypeScript**: For a robust and scalable frontend.
- **Component-Based Architecture**: To ensure modularity and reusability.
- **Tailwind CSS with Custom Theme**: For rapid and consistent styling.
- **Framer Motion**: For smooth and engaging animations.
- **Responsive Design**: Optimized for all devices.
- **Performance Optimization**: Focused on Core Web Vitals.
- **Accessibility Compliance**: Ensuring the site is usable by everyone.

## Current Site Structure to Preserve

### Pages
- Home page (Hero, Plans, Features, Founders sections)
- All Plans page (detailed pricing for different categories)
- Legal pages (Privacy, Terms, Refund)

### Components to Rebuild with New Design
- Header (promotional banner, navigation)
- Hero section (animated elements)
- Plans section (pricing cards)
- Features section (service benefits)
- Founders section (team profiles)
- Footer (links, social media)

### Functionality to Maintain
- Navigation system
- Plan selection and comparison
- Links to external panel (panel.espressohost.xyz)
- Responsive design
- Animations and interactive elements

## Design Direction

- **New Visual Design**: Moving away from the current dark theme with a fresh color palette.
- **Modern Coffee-Themed Elements**: Reimagining branding with contemporary styles.
- **New UI Patterns**: For cards, buttons, and interactive elements.
- **Sophisticated Typography and Spacing**.
- **Modern Design Trends**: Incorporating glassmorphism, neumorphism, or other contemporary styles.

## Content Requirements

- All existing text, pricing, features, plan structures, and team information will be maintained.
- All legal content will be preserved.

## Additional Features to Consider

- Improved animations and transitions
- Enhanced mobile experience
- Better visual hierarchy and information architecture
- Micro-interactions for user engagement
- Improved call-to-action elements
- Light/dark mode toggle

## Setup

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd my-app
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
